# sandfall
A pixel simulation engine, currently Windows only

## How to build and run
1. Build with cmake
2. Copy the res directory into the build directory alongside the executable

The build directory should look like
```sh
|-- res
|   |-- fragment.glsl
|   `-- vertex.glsl
|-- glfw3.dll
|-- sandfall.exe
|-- sandfall.exp
`-- sandfall.lib
```
